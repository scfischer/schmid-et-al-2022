Detailed Description

This class uses the plugin Plotter to plot the oxygen dissociation curve, which is the oxygen saturation of the blood over a range of oxygen partial pressure values.

This class inherits from the MultiPlot class and thus supports plotting results of multiple simulation instances. It receives the datapoints from the Simulation.Systems.Output.OutputCollector of the respective instance. The x-axis (oxygen partial pressure values) is adjusted for changes in the gas unit (AppSettings.gasUnit.
