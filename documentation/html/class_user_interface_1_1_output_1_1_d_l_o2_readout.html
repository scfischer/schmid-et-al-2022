Detailed Description

This class determines the diffusion capacity of the lung for oxygen ("DLO2") from the simulation output. It passes the result to the appropriate display in the UI.

Simulation.Systems.Output.OutputCollector.o2UptakeVolume is extrapolated to the whole lung by multiplication with the number of alveoli in the human lung (480*10^6, Ochs et al., 2004). DLO2 is calculated as oxygen uptake divided by simulation time and the mean gradient of pO2 between alveolar space and the blood.
