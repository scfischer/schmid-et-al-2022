Detailed Description

This class is responsible for tinting the capillary material to represent oxygenation of the blood flowing through.

This is achieved by taking the oxygen partial pressure of the blood in each capillary section, determining a color to represent it, then using that color for the central column of pixels of a section. To achieve a more appealing result, a gradient is calculated for the transition between adjacent sections. The color values are applied dynamically to the main texture of the capillary material.
