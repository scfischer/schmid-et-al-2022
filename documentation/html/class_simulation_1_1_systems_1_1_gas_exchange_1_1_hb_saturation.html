Detailed Description

This class performs the calculations of hemoglobin (Hb) saturation with oxygen as red blood cells progress along the respiratory capillary - i.e. the resulting saturation values are obtained for every capillary section.

O2 binding to and dissociation from hemoglobin is depending on factors like the respective partial pressures of the respiratory gases in the blood pO2BloodArray and pCo2BloodArray, blood temperature bloodTemperature, the pH of red blood cells PHrbc and concentration of the organic phosphate 2,3-DPG in the blood concentrationDpg. The mathematical model used to express these dependencies and respective contstants was adopted from (Dash et al., 2016).

This method performs the key calculations that yield the saturation of hemoglobin with oxygen.

First, the p50 value for the current conditions is obtained from P50(float) and the Hill coefficient nH for the current pO2 is obatined from Hill(float). With these values, hemoglobin oxygen saturation can be calculated in a Hill equation as proposed by (Dash et al., 2016).

Parameters
pO2Partial pressure of oxygen in the blood.
pCo2Partial pressure of carbon dioxide in the blood.
hbO2SatOutput for hemoglobin oxygen saturation.

This method instructs the calculation of hemoglobin oxygen saturation for each capillary section and stores the results in an array.

This method iterates over the values in the arrays and calls CalculateHbO2Saturation(in float,in float,out float) for each set of values. hbO2Saturations[i] is calculated from pO2[i] and pCo2[i].

Parameters
pO2Array of oxygen partial pressure values of all capillary sections.
pCo2Array of carbon dioxide partial pressure values of all capillary sections.
hbO2SaturationsOutput array of hemoglobin oxygen saturation values of all capillary sections.< /param>
Exceptions
ArgumentExceptionThrown when length of O2 and CO2 partial pressure arrays doesn't match.

This method instructs the calculation of hemoglobin oxygen saturation values to be plotted in the oxygen dissociation curve.

It creates an array of numValues evenly distributed oxygen partial pressure values from pO2Min (inclusive) to pO2Max (inclusive). The method CalculateHbO2Saturation(in float, in float, out float) is called to calculate the respective hemoglobin oxygen saturation value from every pO2 value in this array and the current pCO2 of the blood.

Parameters
pCo2Current partial pressure of carbon dioxide in the (incoming) blood.
pO2MinLower limit of the range of oxygen partial pressure values.
pO2MaxUpper limit of the range of oxygen partial pressure values.
numValuesNumber of datapoints, minimum 2 (for pO2Min and PO2Max).
Returns
Array of numValues evenly distributed pO2 values in order from pO2Min to PO2Max and the corresponding hemoglobin oxygen saturation values.
Exceptions
ArgumentExceptionThrown when numValues is less than 2
