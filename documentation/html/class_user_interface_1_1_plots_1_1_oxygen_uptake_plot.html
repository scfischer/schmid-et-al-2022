Detailed Description

This class uses the plugin Plotter to plot the total oxygen uptake summed up over the course of the simulation.

This class inherits from the MultiPlot class and thus supports plotting results of multiple simulation instances. The elapsed time since the start of the simulation is determined. The current values for the total oxygen uptake are received from the Simulation.Systems.Output.OutputCollector of the respective instance.
