Detailed Description

A base class for simulation components that are also MonoBehaviors, searching for an Instance either as a serialized reference or upwards in the hierarchy. It provides easy access to other simulation components through AlveolusController. When SetInstance is called with a new AlveolusController, Listeners are automatically updated, e.g to react to changes in the ParameterData instance. Furthermore, if a ParameterData instance is already set on the AlveolusController, the HandleReset method is called. This allows to react to any kind of initialization by properly implementing HandleReset.
