Detailed Description

This class uses the plugin Plotter to plot the oxygen saturation over the transit time along the representative alveolar capillary.

This class inherits from the MultiPlot class and thus supports plotting results of multiple simulation instances. It receives the datapoints from the Simulation.Systems.Output.OutputCollector of the respective instance.
