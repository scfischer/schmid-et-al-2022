enum Simulation.Systems.Erythrocytes.AssignPath

Controls the erythrocyte's movement along the paths through the representative capillary.

The erythrocyte moves along the paths in the following order: inflow, main, then outflow. As soon as it reaches the end of one path, it is positioned at the start of the next path which it and subsequenty follows. At the end of the main path, ErythrocyteGasExchange is instructed to take stock of oxygen saturation changes across the main capillary. Once it reaches the end of outflow path, it is destroyed.

This class uses the plugin PathCreation as a tool to orientate along a VertexPath. Depending on EndOfPathInstruction/c>, it i.a. offers the possibility to access the position and rotation of a transform travelling a given distance along the given path.

Three different modes that control at which part of the simulation the erythrocyte is and how it has to move.
