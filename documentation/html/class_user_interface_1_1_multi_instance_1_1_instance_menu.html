Detailed Description

This class organizes simultaneous simulation of multiple AlveolusController instances.

It keeps a list of active AlveolusController instances. The length of this list is confined to a limit defined in AppSettings. Here, main functionalities such as creation of new instances, cloning or deletion of existing instances are implemented. Only the 3D model associated to the selected instance is visualized.
