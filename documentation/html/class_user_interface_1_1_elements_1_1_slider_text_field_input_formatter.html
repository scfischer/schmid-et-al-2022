void UserInterface.Elements.SliderTextFieldInputFormatter.UpdateDisplayValue ( float  value)

Updates the InputField this script is attached to with a string representation of the floating point number passed as parameter.

Rounds the result to the nearest integer before string conversion for values of 3 or larger, and creates a string output as number with two decimal places for smaller values.
