Detailed Description

This class controls the user configuration of model parameters related to incoming, deoxygenated blood.

Input from the user is passed to ParametersData of the corresponding instance. It takes into account which pressure unit is currently selected and converts the values accordingly if necessary (the model calculations are all performed in mmHg). If changes are made in ParametersData or another instance is selected, the parameter sliders are updated immediately.
