Detailed Description

This class should be attached to GUI elements that, when hovered over, should cause other objects in the scene to be highlighted. Needs to be assigned to a target for highlighting using the HighlightController, by dragging this trigger and the target containing an HighlightController.highlightAssociations component into the HighlightController listing.
