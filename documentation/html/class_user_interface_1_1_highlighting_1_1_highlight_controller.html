Detailed Description

The class which controls highlighting of objects based on GUI interactions.

It allows assigning pairs of HighlightTriggers and targets that should have highlighting toggles, thus allowing to re-use scripts for triggers without having to modify the code. Instead, the trigger element is assigned to HighlightingAssociation.trigger of highlightAssociations together with its intended target. The HighlightingAssociation.target needs to implement IHighlightable, the methods of which are invoked by this class to turn highlighting on or off.
