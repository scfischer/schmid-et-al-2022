\doxysubsection{Detailed Description}
This class uses the plugin Plotter to plot the oxygen dissociation curve, which is the oxygen saturation of the blood over a range of oxygen partial pressure values. 

This class inherits from the \mbox{\hyperlink{class_user_interface_1_1_plots_1_1_multi_plot}{Multi\+Plot}} class and thus supports plotting results of multiple simulation instances. It receives the datapoints from the \mbox{\hyperlink{class_simulation_1_1_systems_1_1_output_1_1_output_collector}{Simulation.\+Systems.\+Output.\+Output\+Collector}} of the respective instance. The x-\/axis (oxygen partial pressure values) is adjusted for changes in the gas unit (App\+Settings.\+gas\+Unit.
