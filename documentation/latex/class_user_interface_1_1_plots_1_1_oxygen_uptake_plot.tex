\doxysubsection{Detailed Description}
This class uses the plugin Plotter to plot the total oxygen uptake summed up over the course of the simulation. 

This class inherits from the \mbox{\hyperlink{class_user_interface_1_1_plots_1_1_multi_plot}{Multi\+Plot}} class and thus supports plotting results of multiple simulation instances. The elapsed time since the start of the simulation is determined. The current values for the total oxygen uptake are received from the \mbox{\hyperlink{class_simulation_1_1_systems_1_1_output_1_1_output_collector}{Simulation.\+Systems.\+Output.\+Output\+Collector}} of the respective instance.
