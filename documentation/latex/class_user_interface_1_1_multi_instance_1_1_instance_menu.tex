\doxysubsection{Detailed Description}
This class organizes simultaneous simulation of multiple Alveolus\+Controller instances. 

It keeps a list of active Alveolus\+Controller instances. The length of this list is confined to a limit defined in App\+Settings. Here, main functionalities such as creation of new instances, cloning or deletion of existing instances are implemented. Only the 3D model associated to the selected instance is visualized.
