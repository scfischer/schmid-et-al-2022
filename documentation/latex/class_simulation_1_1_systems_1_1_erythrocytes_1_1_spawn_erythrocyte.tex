\doxysubsection{Detailed Description}
Class to spawn new erythrocytes at the beginning of the inflow capillary. 

\mbox{\hyperlink{namespace_simulation_1_1_systems_1_1_erythrocytes}{Erythrocytes}} are spawned on the path of the inflow capillary. This class also stores the respective instances of the following paths (main and outflow), in order to pass them to the \mbox{\hyperlink{class_simulation_1_1_systems_1_1_erythrocytes_1_1_erythrocyte_movement}{Erythrocyte\+Movement}} component of child erythrocytes. Initially, erythrocytes are spread along the path with the appropriate distance resembling the current blood volume, i.\+e. erythrocyte density. As the erythrocytes move, new clones are spawned at the start of the path in appropriate distance to the previous erythrocyte.
