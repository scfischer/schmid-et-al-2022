\doxysubsection{Detailed Description}
This class controls the user configuration of model parameters related to incoming, deoxygenated blood. 

Input from the user is passed to Parameters\+Data of the corresponding instance. It takes into account which pressure unit is currently selected and converts the values accordingly if necessary (the model calculations are all performed in mm\+Hg). If changes are made in Parameters\+Data or another instance is selected, the parameter sliders are updated immediately.
