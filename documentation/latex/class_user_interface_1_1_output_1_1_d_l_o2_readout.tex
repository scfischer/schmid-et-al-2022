\doxysubsection{Detailed Description}
This class determines the diffusion capacity of the lung for oxygen (\char`\"{}\+DLO2\char`\"{}) from the simulation output. It passes the result to the appropriate display in the UI. 

\mbox{\hyperlink{class_simulation_1_1_systems_1_1_output_1_1_output_collector_a0937e895b4fe57c8c67de8aa236e59ef}{Simulation.\+Systems.\+Output.\+Output\+Collector.\+o2\+Uptake\+Volume}} is extrapolated to the whole lung by multiplication with the number of alveoli in the human lung (480$\ast$10$^\wedge$6, Ochs et al., 2004). DLO2 is calculated as oxygen uptake divided by simulation time and the mean gradient of p\+O2 between alveolar space and the blood.
